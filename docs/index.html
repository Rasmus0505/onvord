FEATURES

为 AI Native 时代打造

从底层改变 SOP 的制作和呈现方式，让人和 AI 都能看懂。

🎙️

实时语音流式转录

基于 Deepgram Nova-2 引擎，支持中文、英文及多语种自动检测。边操作边讲解，语音自动关联到对应的点击动作，无需事后剪辑。

🖱️

智能操作捕获

自动记录点击、输入、滚动。精准区分「文字选择」与「点击」，智能过滤点击页面空白处的无意义冗余操作。

📸

醒目操作点标注截图

每次操作自动截取当前屏幕，并用三层复合光圈醒目标注点击/选择位置。无论是浅色还是深色背景页面，焦点皆清晰可见。

📄

单页面图文结构化导出

一键导出为包含 HTML、内联 CSS 和 base64 图片的独立文件。排版精美的深色主题时间线，并在页头附带「文档说明」—— 发给大模型（如 Claude/GPT-4V），AI 瞬间读懂操作意图与确切的 CSS Node 路径。

SCENARIOS

你可以用它做什么？

01

教 AI 做任务

录制一遍业务系统操作流，导出包含选择器的结构化 SOP，喂给 Browser Agent 即可实现自动化执行。

02

产品操作手册

作为 PM 或开发，一边演示一边口述，直接生成图文并茂的用户教程文档，彻底告别繁琐的截图拼图工作。

03

Bug 提交流程

测试同学还原 Bug 现场的终极武器。每一个点击路径、对应的页面状态和操作者的补充说明完美绑定，方便研发排查。

04

团队内部培训

老员工离职交接、新人培训，只需把常规后台操作录一遍，新人即可依据步骤进行自我 onboarding。

QUICK START

三步开启自动化

1. 安装插件

下载源码并在 Chrome 中「加载已解压的扩展程序」。打开侧边栏。

2. 填入 API Key

前往 Deepgram 免费注册获取 API Key（首送 $200 额度），在扩展设置页粘贴保存。

3. 边说边点 > 导出

点击「开始录制」，在任意网页操作并讲话讲解。结束后点击导出，获得结构化的独立 HTML 文件。
